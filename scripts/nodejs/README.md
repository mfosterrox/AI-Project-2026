# GitHub REST / GraphQL collectors

- getting_repos_v2.js — REST search; set CLIENT_ID and CLIENT_SECRET in .env
- githubGraphQLApiCallsDO_V2.js and githubGraphQLApiCallsDO_V3.js — GraphQL enrichment; set TOKEN, START, END in .env

Copy .env.example to .env, run npm install here, or run python scripts/ingest_data.py from the repo root.
